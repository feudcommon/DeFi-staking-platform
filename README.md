# DeFi Staking Platform

A decentralized staking platform built on SCAI Mainnet allowing users to stake ERC20 tokens and earn rewards.

## Live Demo
https://vercel.com/stakingcontract/de-fi-staking-platform

## Tech Stack
- Solidity
- React
- TypeScript
- Ethers.js
- MetaMask / WalletConnect / Coinbase Wallet

## Contracts (SCAI Mainnet)
- Token: 0x4EF03D37c441BcF78D61367f4EE709027632d929
- Staking: 0x9aab06FAE31e082c26979afca9E53897dB57D50C

## Features
- Stake tokens
- Withdraw tokens
- Claim rewards
- View dashboard (staked amount, rewards, APR, APY, wallet balance)
